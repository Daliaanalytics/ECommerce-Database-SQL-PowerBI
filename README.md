# ECommerce-Database-SQL-PowerBI

## Project Overview
A multi-table E-Commerce database built with Python-generated synthetic data,
analyzed with SQL queries and visualized in Power BI.

## Tools Used
-  Python (Faker, Pandas, NumPy) — Data Generation
-  SQL Server (SSMS) — Database & Queries
-  Power BI — Dashboard

## SQL Analysis
- Revenue by Order Status
- Top 10 Customers by Spending
- Top 10 Products by Quantity Sold
- Revenue by Category
- Customers Without Orders (LEFT JOIN)
- Monthly Revenue Trends

## Dashboard Pages
- **Page 1:** Overview — KPIs, Revenue by Status, Monthly Trends, Products & Categories
- **Page 2:** Customers — Top Spenders, Customers Without Orders, Global Map

## Key Findings
-  **Books** is the top revenue category with **$772K** in sales
-  **Brenda Nunez** is the top spending customer with **$8,198**
-  **April 2025** was the best month with **$68,142** in revenue
-  **64 customers** never placed an order
-  **Refunded** orders had the highest revenue at **$288K**
-  **Vision-oriented impactful archive** is the top selling product with **131 units**
